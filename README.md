# Brible

Brible es una aplicación web desarrollada con React + Vite, Express y MongoDB, pensada para los amantes del cine y las series. Permite explorar contenido en tendencia, buscar información de películas, series o actores, y gestionar tu cuenta de manera segura.

### Prueba el sitio
https://movie-library-8w5d.onrender.com

### Tecnologías utilizadas
* Frontend: React, Vite
* Backend: Express, Node.js
* Base de datos: MongoDB
* Seguridad: bcrypt, JWT
* Correo electrónico: Nodemailer
* Tareas programadas: Cron (para limpieza de cuentas inactivas)

### Características

* Exploración de películas, series y personas en tendencia.
* Registro e inicio de sesión seguro con bcrypt y JWT.
* Envío de correos automáticos de confirmación de cuenta.
* Búsqueda de información con paginación.
* Diseño responsivo, compatible con PC y dispositivos móviles.
* Cronjob para eliminar cuentas no confirmadas después de 90 días de su creación.
