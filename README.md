# Brible

Desarrollado con React + Vite, Express y Mongo, Brible es una aplicación para la exploración de contenido cinematográfico en tendencia donde también puedes buscar inforamción de tus peliculas, series o actores favoritos.

## Características:
-Exploración de películas, series y personas en tendencia.
